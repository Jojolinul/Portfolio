RoboHood

Description:

A team of two programmers and six artists are asked to colaborate on the creation of a game in Unreal Engine 4. The end result is a lan multiplayer, one versus one, robot themed shooter in which the players can pick up items (health, bombs, rockets), use launchpads and teleporters, pick from a variety of maps and characters, and see their success (or failure) on the scoreboard. A spider AI can be seen on one of the maps.

SoundLabyrinth

Description:

For my final year project I wanted to do something challenging so I decided to create a demo that consisted of a custom built OpenGL engine that supports graphics shaders, ray tracing, vertex manipulation, lighting, sound manipulation, gamestates, file read / write functionality (both binary and text files). Most of these topics were not taught in university thus I had to do a lot of research for this project to come to life.

Cube.AI

Description:

Cube.AI is a really fast paced one versus one, top-down view, rescue the hostage type of game. It is made in the Unity Engine and it is using the Post Processing Stack to give it extra depth and beauty in the graphical department. The controls are extremely simple as is the goal of the game but because of the competitive side of it mistakes will be made by each side. This project is currently being revamped to an Android version that I intend to release in 2018 on the Play Store as "Cube.AI".
