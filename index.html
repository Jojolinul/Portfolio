TowerWrite

Description:

TowerWrite was a project made for the C++ Game Development module where the graphical aspect was not important but rather the complexity of the code, pointer management, code structure, use of libraries such as SDL, and showing our C++ knowledge. Therefore TowerWrite was born, a tower defense game where you control one tower and can click on the screen and type what you want the tower to do (i.e "shoot" or "cow"). The zombies travel on a path that is generated using a text file that contains waypoints; these waypoints are also used for the creation of the path graphics.
